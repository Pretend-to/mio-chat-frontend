# Mio-Chat 前端设置页面需求文档 (PRD)

## 1. 个人资料 (Profile)
*   **用户昵称 (User Name)**: 
    *   允许用户修改显示的名称。
    *   **影响范围**: 移动端 Header 的欢迎语、聊天记录中用户侧的显示名称。
*   **个性头像 (Avatar)**:
    *   支持上传本地图片作为用户头像。
    *   **影响范围**: 整个应用的头像展示。

## 2. 聊天偏好 (Chat Settings)
*   **发送快捷键 (Send Shortcut)**:
    *   选项 A: `Enter` 发送，`Ctrl/Cmd + Enter` 换行 (默认)。
    *   选项 B: `Ctrl/Cmd + Enter` 发送，`Enter` 换行。
    *   **目的**: 适应不同用户的输入习惯。
*   **消息气泡样式**: 
    *   提供几种预设样式（如：现代简约、经典 QQ 感、无边框感）。
*   **字体大小**:
    *   滑块调节聊天内容字号。

## 3. 外观与主题 (Appearance)
*   **暗黑模式 (Dark Mode)**:
    *   跟随系统 / 强制开启 / 强制关闭。
*   **自定义背景 (Custom Background)**:
    *   允许用户上传背景图片，并调节模糊度 (Blur) 和透明度 (Opacity)。
*   **毛玻璃效果开关**:
    *   如果设备性能较弱，允许关闭全局的 `backdrop-filter` 效果以提升流畅度。

## 4. 存储与数据 (Data Management)
*   **缓存清理 (Clear Cache)**:
    *   一键调用 Service Worker `CLEAR_IDB_CACHE` 逻辑，清理缓存。
*   **本地数据导出/导入**:
    *   将 `localStorage` 中的联系人、聊天记录和设置导出为 JSON 文件，实现多端同步或备份。

## 5. 交互与通知 (Interaction)
*   **消息提示音**:
    *   开关控制发送/接收消息时是否播放提示音。

---

### 优先级建议 (P0):
1. 用户昵称修改
2. 快捷键配置
3. 自定义背景
4. 缓存清理
